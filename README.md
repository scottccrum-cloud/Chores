# Chores - Kid Chore Chart

A simple, user-friendly web application designed to help track and manage daily chores for the family. This project provides individual personalized dashboards for the kids to view their tasks, alongside a dedicated parent portal for oversight.

## 🚀 Live Demo
You can view the live, deployed project here:
[Chores Live Dashboard](https://scottccrum-cloud.github.io/Chores/)

---

## 📋 Features & Structure

The repository consists of static HTML pages tailored for each family member:

* **`index.html`**: The main landing hub/dashboard where family members can select their profile.
* **`parent.html`**: The parental control view to manage, assign, and review chore progress.
* **`ethan.html`**: Personalized chore checklist for Ethan.
* **`jacob.html`**: Personalized chore checklist for Jacob.
* **`mia.html`**: Personalized chore checklist for Mia.

---

## 🛠️ Built With

* **HTML5** - Core structure and page layouts.
* **GitHub Pages** - Free automated hosting and deployment.
