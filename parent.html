How to Publish (Free)

Step 1: Create a free Firebase project at console.firebase.google.com → Add project → Realtime Database → Start in test mode. Copy the DB URL.

Step 2: Upload all 4 HTML files (jacob.html, ethan.html, mia.html, parent.html) to GitHub Pages (free). Your URLs will be:
https://yourusername.github.io/chores/jacob.html

Step 3: Open parent.html, enter your DB URL + Family Code, click Save. Give kids their page URL + the Family Code + DB URL to enter on first launch.

Tip: Bookmark each child's page and add to their iPhone Home Screen: Share → Add to Home Screen.
